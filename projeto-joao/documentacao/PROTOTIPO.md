# Protótipo e identidade visual

O protótipo foi implementado diretamente como interface navegável. Identidade: painel espacial industrial escuro, verde ácido para ações, tipografia de alta legibilidade e estados visuais de operação/alerta.

Fluxo: visão geral → máquinas → cadastro → gestão integrada → produção → ocorrências. Há link de salto, regiões semânticas, mensagens `aria-live`, foco nativo e adaptação para telas menores.
