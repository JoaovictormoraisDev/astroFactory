# Protótipo e identidade visual

O protótipo navegável usa linguagem editorial clara, azul/verde para operação sustentável e cartões responsivos. Fluxo: visão geral → máquinas → impacto → cadastro → gestão integrada → produção → SST.

Acessibilidade: HTML semântico, link de salto, mensagens de estado, contraste, rótulos de formulário e redução de movimento.
