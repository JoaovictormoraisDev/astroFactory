# Escopo — PitecoFlow

O PitecoFlow centraliza máquinas, produção, indicadores e ocorrências de SST para operadores e supervisores industriais. A entrega inclui CRUD de máquinas, produção, produtividade, dashboard e registro preventivo de riscos.

Arquitetura: interface responsiva HTML/CSS/JavaScript → API REST Node.js/Express → MySQL. O projeto utiliza Git e testes automatizados com `node:test`.
